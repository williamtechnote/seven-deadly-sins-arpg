# 七宗罪像素风动作RPG — 游戏设计文档

## 概述

一款运行在浏览器中的像素风等距45度视角动作角色扮演游戏。玩家操控主角挑战以七宗罪为主题的七大Boss，通过多武器切换、闪避和策略搭配来征服每一场Boss战。游戏包含中心枢纽探索、NPC交互、物品系统和武器升级。

## 技术选型

- **引擎**：Phaser.js（CDN引入）
- **架构**：单文件快速原型，无构建工具，浏览器直接运行
- **视角**：等距45度（Isometric），地图用 Tiled 编辑器制作导出 JSON
- **美术**：itch.io 等平台的免费像素素材包
- **存档**：浏览器 localStorage

## 项目结构

```
amy_project/
├── index.html          # 入口文件，加载 Phaser CDN
├── game.js             # 主游戏文件，所有场景和核心逻辑
├── data.js             # 数据配置（Boss、武器、对话、物品）
└── assets/             # 美术资源
    ├── sprites/        # 角色精灵图
    ├── tilesets/       # 地图贴图
    ├── portraits/      # 对话立绘
    └── ui/             # UI 元素
```

## 场景系统（Phaser Scenes）

| 场景 | 职责 |
|------|------|
| BootScene | 加载所有资源，显示loading画面 |
| TitleScene | 主菜单（开始游戏、继续、设置） |
| HubScene | 中心枢纽城镇，NPC交互、武器切换、进入关卡入口 |
| LevelScene | 通用关卡场景，根据参数加载不同Boss区域的地图和小怪 |
| BossScene | Boss战专用场景，加载Boss机制和战斗逻辑 |
| DialogScene | 对话系统覆盖层，可叠加在任何场景之上 |
| UIScene | 常驻HUD（血条、武器栏、道具快捷键），平行运行 |

## 战斗系统

### 资源系统

- **HP（血量）**：被攻击消耗，药水回复
- **Stamina（体力）**：攻击、闪避、特殊技能消耗体力，随时间自动恢复

### 核心机制

- **闪避**：翻滚有短暂无敌帧（约0.3秒），冷却时间约1秒
- **击退与硬直**：攻击造成击退效果，被击中有短暂硬直
- **掉落物**：小怪掉落金币和随机道具，Boss掉落特殊武器/材料

### 武器系统（5种）

| 武器 | 攻击方式 | 特殊技能 | 特点 |
|------|---------|---------|------|
| 长剑 | 近距离横扫 | 蓄力斩（高伤害扇形） | 初始武器，均衡 |
| 双刃 | 快速连击 | 旋风斩（AOE） | 高攻速低单次伤害 |
| 大锤 | 慢速重击 | 地震冲击（范围眩晕） | 高伤害慢速度 |
| 弓箭 | 远程射击 | 箭雨（范围落点） | 远程安全输出 |
| 法杖 | 魔法弹 | 元素爆发（大范围魔法） | 高体力消耗，高范围伤害 |

### 武器升级

- 每把武器3级：基础 → 强化 → 极致
- 升级提升伤害和特殊技能效果
- 需要金币 + 对应Boss材料

## 七宗罪Boss设计

| 罪 | Boss名 | 视觉主题 | 核心机制 | 区域风格 |
|----|--------|---------|---------|---------|
| 傲慢 Pride | 堕天骑士 | 金色铠甲、光翼 | 镜像分身，复制玩家动作反击 | 天空神殿 |
| 嫉妒 Envy | 影形者 | 暗绿色流体，不断变形 | 偷取玩家当前武器能力攻击 | 毒沼迷宫 |
| 暴怒 Wrath | 炎魔将军 | 红色巨人，全身火焰 | 狂暴模式：血量越低攻击越猛越快 | 熔岩锻炉 |
| 懒惰 Sloth | 梦境蛛后 | 紫色巨蛛，飘浮催眠雾 | 减速领域+召唤小蛛，玩家被debuff减速 | 沉睡森林 |
| 贪婪 Greed | 黄金龙王 | 金色龙，身上镶满宝石 | 地面铺满金币陷阱，宝箱诱骗玩家 | 宝藏地窟 |
| 暴食 Gluttony | 深渊巨口 | 肥胖恶魔，巨大嘴巴 | 吞噬小怪变强，需抢先击杀小怪 | 腐烂宴厅 |
| 色欲 Lust | 魅惑女妖 | 妖艳人形，蝴蝶翅膀 | 魅惑弹——命中后操控反转 | 幻梦花园 |

### Boss战通用结构（3阶段）

- **阶段1（100%-60% HP）**：基础攻击模式，让玩家学习Boss行为
- **阶段2（60%-30% HP）**：激活核心机制，难度提升，新增招式
- **阶段3（30%-0% HP）**：全力爆发，所有招式加强，场地变化

### Boss战流程

进入Boss房间 → 直接开始战斗 → 击败后触发剧情对话（揭示Boss故事） → 获得奖励 → 返回Hub

### 小怪

- 每个Boss区域有3-4种主题小怪
- 小怪AI：巡逻 → 发现玩家 → 追击 → 攻击
- 清完小怪或穿过区域后进入Boss房间

## Hub中心枢纽 — "净罪庇护所"

安全区域，没有战斗，包含以下功能点：

- **武器台**：切换/升级武器
- **商店NPC**：用金币购买药水、材料
- **铁匠NPC**：用Boss掉落材料强化武器
- **剧情NPC**：提供世界观线索、支线对话
- **七道传送门**：通往七个Boss区域的入口，击败Boss后门上出现对应标记

## 物品系统

| 类型 | 示例 | 用途 |
|------|------|------|
| 消耗品 | 生命药水、体力药水 | 快捷栏1-4使用，战斗中即时回复 |
| 材料 | Boss掉落的罪之结晶 | 铁匠处强化武器 |
| 金币 | 小怪/宝箱掉落 | 商店购买消耗品和材料 |
| 关键道具 | 击败Boss获得的罪之印记 | 剧情推进道具，集齐七枚触发最终章 |

### 背包界面（Tab键）

- 分页显示：武器 / 消耗品 / 材料 / 关键道具
- 简洁的像素风网格布局

## 关卡探索

- 每个Boss区域有2-3个房间的探索路径，包含小怪战斗和宝箱
- 通关后可重复进入刷材料和金币
- 区域内无存档点，死亡回到Hub

## UI/HUD

### 战斗HUD（常驻）

- **左上角**：HP条（红色）+ Stamina条（黄色）+ 角色小头像
- **左下角**：当前武器图标 + Q/E切换提示
- **右下角**：快捷栏道具（1-4格）
- **右上角**：当前区域名称 + 小地图（仅Hub中显示）

### 对话界面

- 屏幕下方1/3区域的对话框
- 左侧显示说话者像素立绘
- 右侧文字逐字显示，按任意键推进
- 有选项时显示选择分支

### 暂停菜单（Esc）

- 继续游戏 / 背包 / 武器信息 / 设置 / 返回主菜单

## 存档系统

- 使用浏览器 localStorage 自动存档
- 存档时机：返回Hub时自动保存
- 存档内容：已击败Boss列表、武器等级、物品、金币
- 主菜单可"继续游戏"读取存档

## 游戏流程

```
主菜单 → Hub庇护所 → 选择传送门 → 关卡探索(2-3房间小怪)
→ Boss房间(纯战斗) → 击败后剧情对话 → 获得奖励 → 返回Hub
→ 升级武器/补给 → 下一个传送门 → ...
→ 集齐七枚罪之印记 → 最终Boss解锁 → 通关
```

### 最终Boss

- 集齐七枚罪之印记后，Hub中心出现第八道门
- 最终Boss融合了七宗罪的部分机制，作为全游戏的终极考验
- 击败后播放结局剧情
