# 七宗罪 | Seven Deadly Sins

基于 Phaser 3 的浏览器像素风动作 RPG。操控主角挑战以七宗罪为主题的七大 Boss，通过多武器切换、闪避翻滚和策略搭配征服每一场战斗。

## 快速开始

无需安装依赖，启动本地 HTTP 服务器即可运行：

```bash
# Python
python3 -m http.server 8000

# 或 Node.js
npx serve .
```

浏览器打开 `http://localhost:8000`。

## 操作指南

| 按键 | 操作 |
|------|------|
| WASD | 八方向移动 |
| J / 鼠标左键 | 普通攻击 |
| K / 鼠标右键 | 特殊攻击 |
| Space | 闪避翻滚（无敌帧） |
| Q / E | 切换武器 |
| 1-4 | 使用快捷栏道具 |
| Tab | 背包 |
| H | 操作指引 |
| Esc | 暂停菜单 |
| F | NPC 交互 |

> 游戏内随时按 **H** 可查看完整操作指引。

## 游戏内容

### 五种武器

| 武器 | 类型 | 特点 |
|------|------|------|
| 长剑 | 近战 | 均衡，初始武器 |
| 双刃 | 近战 | 高攻速连击 |
| 大锤 | 近战 | 高伤害重击 |
| 弓箭 | 远程 | 安全输出 |
| 法杖 | 远程 | 范围魔法 |

每把武器可升级 3 级，需要金币 + Boss 掉落材料。

### 七宗罪 Boss

| 罪 | Boss | 区域 |
|----|------|------|
| 傲慢 | 堕天骑士 | 天空神殿 |
| 嫉妒 | 影形者 | 毒沼迷宫 |
| 暴怒 | 炎魔将军 | 熔岩锻炉 |
| 懒惰 | 梦境蛛后 | 沉睡森林 |
| 贪婪 | 黄金龙王 | 宝藏地窟 |
| 暴食 | 深渊巨口 | 腐烂宴厅 |
| 色欲 | 魅惑女妖 | 幻梦花园 |

集齐七枚罪之印记后解锁最终 Boss。

### Hub — 净罪庇护所

安全区域，包含：
- **铁匠** — 升级武器
- **商人** — 购买药水和材料
- **贤者** — 世界观剧情对话
- **七道传送门** — 通往各 Boss 区域

## 游戏流程

```
主菜单 → 净罪庇护所 → 选择传送门 → 关卡探索（小怪战斗）
→ Boss 战 → 剧情对话 → 获得奖励 → 返回庇护所
→ 升级武器 / 补给 → 下一个传送门 → ……
→ 集齐七枚罪之印记 → 最终 Boss → 通关
```

## 项目结构

```
├── index.html      # 入口，加载 Phaser CDN
├── game.js         # 全部场景与核心逻辑
├── data.js         # 数据配置（武器、Boss、NPC、物品）
└── assets/         # 像素美术资源
    └── sprites/    # 角色、敌人、NPC、道具精灵图
```

## 技术栈

- **Phaser 3.80.1**（CDN 引入，无需构建）
- 纯 JavaScript，浏览器直接运行
- localStorage 自动存档
