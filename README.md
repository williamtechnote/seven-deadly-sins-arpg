# 七宗罪 | Seven Deadly Sins

基于 Phaser 3 的浏览器像素风动作 RPG。操控主角挑战以七宗罪为主题的七大 Boss，通过多武器切换、闪避翻滚和策略搭配征服每一场战斗。

## 快速开始

无需安装依赖，启动本地 HTTP 服务器即可运行：

```bash
# Python
python3 -m http.server 8000

# 或 Node.js
npx serve .
```

浏览器打开 `http://localhost:8000`。

## 操作指南

| 按键 | 操作 |
|------|------|
| WASD | 八方向移动 |
| I / J / K / L | 键盘双轴瞄准（松开时保留上次朝向） |
| U / 鼠标左键 | 普通攻击 |
| O / 鼠标右键 | 特殊攻击 |
| Space | 闪避翻滚（无敌帧） |
| Q / E | 切换武器 |
| 1-4 | 使用快捷栏道具 |
| Tab | 背包（点击消耗品自动装入快捷栏） |
| H | 操作指引 |
| Esc | 暂停菜单 |
| F | NPC / 事件房交互 |

> 游戏内随时按 **H** 可查看完整操作指引。

## 游戏内容

### 五种武器

| 武器 | 类型 | 攻击模式 | 特点 |
|------|------|----------|------|
| 长剑 | 近战 | 横扫 | 均衡，初始武器；特攻附带流血 |
| 双刃 | 近战 | 突刺 | 高攻速连击 |
| 大锤 | 近战 | 砸击 | 高伤害重击；特攻附带减速 |

| 法杖 | 远程 | 魔法 | 范围魔法攻击 |

每把武器可升级 3 级（需金币 + Boss 掉落的绑定精华），升级影响伤害、攻速和体力消耗。

| 武器 | 绑定材料 |
|------|----------|
| 长剑 | 贪婪之精华 |
| 双刃 | 傲慢之精华 |
| 大锤 | 暴怒之精华 |
| 弓箭 | 嫉妒之精华 |
| 法杖 | 懒惰之精华 |

每个 Boss 拥有 3 个阶段，随血量降低解锁新招式。集齐七枚罪之印记后解锁最终 Boss **原罪之主**。

### Hub — 净罪庇护所

安全区域，包含：

- **铁匠** — 升级武器 + 制作战斗药剂（净化药剂 / 狂战油）
- **商人** — 购买药水和材料
- **贤者** — 世界观剧情对话
- **七道传送门** — 通往各 Boss 区域
- **Hub 小地图** — 显示房间、传送门、NPC 和玩家位置

### 战斗系统

- **状态效果**：`burn`（持续灼烧）、`bleed`（持续流血）、`slow`（移速降低）——武器特攻和部分敌人技能均可施加
- **Run Modifiers**：每局随机抽取 3 个词缀，影响敌人血量/速度、玩家伤害/承伤、掉落率等
- **挑战目标**：每局随机生成击败敌人数挑战，HUD 实时显示进度，完成后获得金币奖励
- **事件房**：每局关卡中随机生成 1 个事件点，靠近祭坛按 F 打开抉择面板
  - **赌徒圣坛** — 生命换金币
  - **疗愈泉眼** — 高额治疗 / 治疗+净化
  - **血契祭坛** — 本局增伤但提高承伤（激进/稳健两档）
  - **战备商柜** — 花金币购买药剂
  - **祈愿圣坛** — 体力恢复强化 / 特攻冷却缩短二选一

### 经济系统

- 普通小怪掉落金币、药水、低概率区域精华
- 掉落物需走过拾取，带浮字反馈
- 铁匠可制作净化药剂和狂战油

## 游戏流程

```
主菜单 → 净罪庇护所 → 选择传送门 → 关卡探索（小怪战斗 + 事件房）
→ Boss 战 → 剧情对话 → 获得奖励 → 返回庇护所
→ 升级武器 / 补给 → 下一个传送门 → ……
→ 集齐七枚罪之印记 → 最终 Boss → 通关
```

## 技术栈

- **Phaser 3.80.1**（CDN 引入，无需构建）
- 纯 JavaScript，浏览器直接运行
- localStorage 自动存档
- 内置 WebAudio 程序化音效（攻击 / 受击 / UI），无需外部音频资源

## CLI 回归检查

```bash
node scripts/regression-checks.mjs
```

51 项检查覆盖：

- 武器属性成长单调性与绑定材料校验
- 战斗数值基线（长剑攻击范围、小怪移速、玩家死亡冻结）
- 存档序列化 / 反序列化完整性
- 状态效果配置与 tick 伤害逻辑
- 消耗品恢复结算与"已满不消耗"规则
- Boss HUD（血条残影、阶段标签、预警条、破招高亮）
- 键盘瞄准、HUD 提示与快捷栏交互
- HUD 侧栏响应式布局与文本宽度钳制
- 事件房抉择、结算、HUD 摘要与祭坛标签回退
- Run Modifier 抽取唯一性与铁匠制作配方
